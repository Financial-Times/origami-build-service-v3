API Reference

JavaScript

Method GET
Path /v3/bundles/js
Querystring
modules
Required

A comma-separated list of Origami components in the format name@version. The name can be any Origami component and the version is any SemVer compatible version.

source
Required

The systemcode of the system that is is making use of the Origami Build Service.

minify
Defaults to on

Set to on to have the response be minified.

Set to off to have the response not be minified.

esmodules
Defaults to off

Set to on to have the response be in ESM format.

Set to off to have the response be in UMD format.

ua
Internal use only

Set to a normalised useragent string to have the JavaScript be compiled to the latest version of JavaScript that the useragent supports.

This is used internally in the service and is exposed to simplify debugging a response and to acheive a simpler caching workflow with our CDN.

registry
Defaults to bower

Set to bower to have the modules be fetched via the Origami Bower registry.

Set to npm to have the modules be fetched via the npm registry.
Headers
User-Agent

If no ua querystring value is configured, the User-Agent header's value will be used for JavaScript compilation targeting instead.

Accept-Encoding

If no compression querystring value is configured, the Accept-Encoding header's value will be used to decide if compression should be used on the response.
